## üîë Key Concepts Explained

### DAO (Decentralized Autonomous Organization)
The governance layer that enables democratic decision-making:
- **Token-based voting**: EUS tokens grant voting rights
- **Weighted by contribution**: Karma score multiplies voting power
- **Proposal system**: Submit, discuss, vote, execute
- **Transparency**: All decisions recorded on blockchain

**Covered in**:
- [Architecture Doc](./v2.0_kosymbiosis_digitale.md#ai-governance-layers)
- [System Diagrams](./v2.0_system_diagrams.md#21-dao-governance-flow)
- [Use Case Guide](./v2.0_use_case_guide.md#governance-use-cases)

---

### TotallyFruttiKarma (TFK)
The karma-based incentive system that aligns behavior with ecosystem values:
- **Accumulation**: Positive contributions earn karma
- **Benefits**: Enhanced voting power, priority access, rewards
- **Self-Correction**: Continuous Assurance Cycle (CAC) ensures fairness
- **Transparency**: All karma calculations auditable

**Covered in**:
- [Architecture Doc](./v2.0_kosymbiosis_digitale.md#3-totallyfruttikarm-tfk-protocol)
- [System Diagrams](./v2.0_system_diagrams.md#22-ethical-correction-dasseet-ecd-monitoring)
- [Use Case Guide](./v2.0_use_case_guide.md#use-case-51-new-member-journey)

---

### Ethical Correction Dasseet (ECD)
The ethical guardian that monitors and corrects the ecosystem:
- **Real-time Monitoring**: Continuous tracking of all operations
- **Automated Validation**: NSR, OLF, TFK checks on every action
- **Red Code System**: Emergency halt for ethical violations
- **Continuous Assurance Cycle**: Proactive identification of drift

**Covered in**:
- [Architecture Doc](./v2.0_kosymbiosis_digitale.md#2-ethical-correction-dasseet-ecd)
- [System Diagrams](./v2.0_system_diagrams.md#22-ethical-correction-dasseet-ecd-monitoring)
- [Use Case Guide](./v2.0_use_case_guide.md#ethical-correction-use-cases)

---

### Blockchain Integration
The immutable, transparent ledger layer:
- **KarmaBond**: Trust-based bonding with automatic treasury allocation
- **Sustainment**: $10,000 minimum treasury with monitoring
- **EUSDaoGovernance**: Token-based voting rights
- **TrustlessFundingProtocol**: Milestone-based project funding

**Covered in**:
- [Architecture Doc](./v2.0_kosymbiosis_digitale.md#blockchain-integration-layers)
- [System Diagrams](./v2.0_system_diagrams.md#23-karmabond-and-treasury-integration)
- [Use Case Guide](./v2.0_use_case_guide.md#economic-sustainability-use-cases)

Modules for user interaction and system integration:
- **OV (Open Visual)**: Facial recognition authentication portal
