Explain vacuum bridge 


Vacuum Bridge – Overview
A vacuum bridge (also called a vacuum gap or vacuum diode gap) is a very thin region of empty space that separates two conductive surfaces—typically a cathode and an anode. Because there is no intervening material, the only barrier an electron sees is the electric potential formed by the work functions of the electrodes and any externally applied voltage.

Why It Matters
Barrier to charge transport: In classical terms, an electron needs kinetic energy equal to or greater than the barrier height to cross the gap.
Quantum tunneling: Even when the electron’s energy is lower than the barrier, its wave function extends into the vacuum, allowing a finite probability of transmission. This enables current flow at voltages well below the classical “turn‑on” voltage.
Field enhancement: Applying a strong electric field tilts the potential, thinning the barrier and dramatically increasing tunneling probability (Fowler–Nordheim tunneling).

Design Considerations
Gap stability: Maintaining a uniform sub‑nanometer gap requires precision fabrication (e.g., atomic‑layer deposition, focused ion beam patterning) and vibration isolation.
Surface cleanliness: Contaminants alter the effective work function and can introduce dielectric layers that suppress tunneling.
Field enhancement structures: Sharp tips or nanostructured cathodes increase local electric fields, reducing the required macroscopic voltage.
Space‑charge effects: At high current densities, emitted electrons can create a cloud that repels additional electrons, limiting current; designs may incorporate gating electrodes to mitigate this.
Summary
A vacuum bridge is simply a tiny region of empty space separating electrodes. While classically it would block electron flow unless the electrons have enough kinetic energy, quantum mechanics allows tunneling through the barrier, and strong electric fields enable field emission. These effects underpin a wide range of modern technologies—from atomic‑scale imaging to fast, low‑power electronic switches.

What “gap” Means in This Context
Physical vacuum bridge – the literal distance d between two electrodes (often 0.2 nm – 10 nm for tunneling applications).

Integrating Gap‑Reduction and Leverage‑Increase in a Design Cycle
Map the current barrier – quantify gap size (d) and baseline field (E) for a physical device, or list procedural/financial obstacles for a social program.
Identify low‑effort leverage points – raise voltage modestly, add a field‑enhancing tip, or secure a small matching contribution.

By systematically tightening the gap and then applying the strongest feasible leverage, designers achieve maximum performance for both vacuum‑bridge devices and social‑impact initiatives.

The Golden Rule – “Do unto others as you would have them do unto you”
Core Idea
Treat others the way you would wish to be treated. In practice it means maximizing benefit while minimizing harm for the people you interact with.

Design principle	Reduce gap width d and increase field E to raise transmission probability without wasteful energy loss.	Reduce the “social gap” (red tape, information asymmetry) and increase “leverage” (trust, matching resources) to raise impact without exploiting or over‑burdening the recipients.

SROI component	Golden‑Rule lens

By weaving the Golden Rule into both the physical design of a vacuum bridge and the economic evaluation of its societal analogues, we achieve solutions that are effective, efficient, and ethically sound.

# src/vacuum_bridge/physics.py
import math
import argparse
from typing import Tuple

hbar = 1.054_571_817e-34   # J·s
e    = 1.602_176_634e-19   # C
m_e  = 9.109_383_56e-31    # kg

def transmission_probability(d_nm: float,
                              phi_eV: float,
                              V_volts: float = 0.0) -> float:
    """
    T ≈ exp[ -2 d sqrt(2 m (phi - e V)) / ħ ]
    d  – gap width in nanometres
    phi – work‑function (eV)
    V   – applied bias (V); V > 0 lowers the barrier.
    """
    d = d_nm * 1e-9                       # convert to metres
    phi_J = phi_eV * e
    barrier = max(phi_J - e * V_volts, 0.0)   # never negative
    exponent = -2.0 * d * math.sqrt(2 * m_e * barrier) / hbar
    return math.exp(exponent)


def fowler_nordheim_current(d_nm: float,
                            phi_eV: float,
                            V_volts: float,
                            area_m2: float = 1e-12) -> Tuple[float, float]:
    """
    Returns (current_density_A_per_m2, total_current_A).
    Uses the classic Fowler‑Nordheim expression.
    """
    # Local electric field
    E = V_volts / (d_nm * 1e-9)          # V/m

    # Constants from Fowler‑Nordheim (A, B in SI)
    A = 1.54e-6   # A·eV·V⁻²
    B = 6.83e9    # V·eV⁻³⁄²·m⁻¹

    J = (A * E**2 / phi_eV) * math.exp(-B * phi_eV**1.5 / E)  # A/m²
    I = J * area_m2
    return J, I


def cli() -> None:
    parser = argparse.ArgumentParser(
        prog="vacuum-bridge",
        description="Compute tunnelling probability and Fowler‑Nordheim current."
    )
    parser.add_argument("-d", "--gap", type=float, required=True,
                        help="Gap width in nm")
    parser.add_argument("-p", "--phi", type=float, required=True,
                        help="Work function in eV")
    parser.add_argument("-V", "--voltage", type=float, default=0.0,
                        help="Applied bias in volts")
    parser.add_argument("-A", "--area", type=float, default=1e-12,
                        help="Emitting area in m² (default 1 µm²)")
    args = parser.parse_args()

    T = transmission_probability(args.gap, args.phi, args.voltage)
    J, I = fowler_nordheim_current(args.gap, args.phi, args.voltage, args.area)

    print(f"Tunnelling probability   : {T:.3e}")
    print(f"Current density (A/m²)   : {J:.3e}")
    print(f"Total current (A)         : {I:.3e}")

# src/vacuum_bridge/sroi.py
import argparse

def sroi(value_created: float, investment: float) -> float:
    """
    Simple SROI = value_created / investment.
    Both arguments must be given in the same monetary unit.
    """
    if investment == 0:
        raise ZeroDivisionError("Investment cannot be zero.")
    return value_created / investment


def cli() -> None:
    parser = argparse.ArgumentParser(
        prog="sroi-calc",
        description="Calculate Social Return on Investment."
    )
    parser.add_argument("-v", "--value", type=float, required=True,
                        help="Monetised social outcomes (e.g., $) ")
    parser.add_argument("-i", "--investment", type=float, required=True,
                        help="Monetary investment (same currency as value)")
    args = parser.parse_args()

    ratio = sroi(args.value, args.investment)
    print(f"SROI = {ratio:.2f}   ( {args.value:.2f} / {args.investment:.2f} )")
