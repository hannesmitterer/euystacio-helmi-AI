## 🌐 Overview

The **Wall of Entropy** is a transparent public log that documents unauthorized or unethical access attempts, security events, NSR violations, and governance actions affecting this repository. It serves as a living record of the framework's integrity and an educational resource for the global community.

All entries are:
- **Public** — visible to anyone without authentication
- **Immutable** — logged with cryptographic timestamps where possible
- **Non-retaliatory** — contributors who report events are protected under the NSR
- **Educational** — each entry includes learnings and corrective actions

## 🛡️ Reporting an Event

Any participant may report an event to be logged in the Wall of Entropy:

1. **Open an Issue** with the label `WALL-OF-ENTROPY`
2. Include the event details following the log format above
3. The Seedbringer Council will review, anonymize if requested, and add the entry
4. All reporters are protected under the NSR — no retaliation is permitted

For **urgent security events**, follow the process in [SECURITY_NOTES.md](SECURITY_NOTES.md) before opening a public issue.
