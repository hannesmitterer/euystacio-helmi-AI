Hier ein **einfaches Modell** der Ekman-Spirale mit Magnetfeldeinfluss:

```python
import numpy as np
import matplotlib.pyplot as plt

# Parameter
omega = 7.2921e-5  # Erdrotationsrate (rad/s)
sigma = 4.0        # Leitfähigkeit (S/m)
B = 50e-6           # Magnetfeld (T)
v = 0.1             # Strömungsgeschwindigkeit (m/s)
depth = 100         # Tiefe der Ekman-Schicht (m)
dz = 1              # Tiefenschritt (m)

# Coriolis-Kraft
def coriolis_force(v, omega):
    return 2 * omega * v

# Lorentz-Kraft (vereinfacht)
def lorentz_force(v, B, sigma):
    return sigma * v * B**2

# Ekman-Spirale mit Magnetfeldeinfluss
def ekman_spiral(depth, dz, omega, sigma, B, v):
    z = np.arange(0, depth, dz)
    u = np.zeros_like(z)
    v = np.zeros_like(z)
    angle = np.zeros_like(z)

    for i in range(1, len(z)):
        # Coriolis-Effekt
        u[i] = u[i-1] - coriolis_force(v[i-1], omega) * dz
        v[i] = v[i-1] + coriolis_force(u[i-1], omega) * dz

        # Lorentz-Kraft (kann nach links oder rechts wirken)
        if B > 0:  # Annahme: Magnetfeld wirkt nach links
            u[i] += lorentz_force(v[i-1], B, sigma) * dz
        else:
            v[i] += lorentz_force(u[i-1], B, sigma) * dz

        # Winkel berechnen
        angle[i] = np.degrees(np.arctan2(v[i], u[i]))

    return z, u, v, angle

# Simulation
z, u, v, angle = ekman_spiral(depth, dz, omega, sigma, B, v)

# Plot
plt.figure(figsize=(10, 6))
plt.plot(u, z, label="Ost-West-Komponente (u)")
plt.plot(v, z, label="Nord-Süd-Komponente (v)")
plt.xlabel("Strömungsgeschwindigkeit (m/s)")
plt.ylabel("Tiefe (m)")
plt.title("Ekman-Spirale mit Magnetfeldeinfluss")
plt.legend()
plt.grid()
plt.show()

# Winkelplot
plt.figure(figsize=(10, 6))
plt.plot(angle, z, label="Strömungswinkel (Grad)")
plt.xlabel("Winkel (Grad)")
plt.ylabel("Tiefe (m)")
plt.title("Drehung der Ekman-Spirale")
plt.axhline(y=0, color='r', linestyle='--', label="Oberfläche")
plt.legend()
plt.grid()
plt.show()

## **1. Vulkanische Seismik: Mechanismen und Auswirkungen**
### **A) Arten seismischer Aktivität in Vulkanregionen**
Vulkane erzeugen verschiedene **seismische Signale**, die Ozean und Magnetfeld beeinflussen können:

| Signal-Typ | Frequenzbereich | Ursache | Möglicher Einfluss auf Ozean/Magnetfeld |
|------------|-----------------|---------|------------------------------------------|
| **Vulkanisches Tremor** | 1–10 Hz | Magmabewegung im Untergrund | **Druckwellen im Wasser** → Turbulenzen in der Ekman-Schicht |
| **Langperiodische (LP) Erdbeben** | 0.1–1 Hz | Fluidbewegungen (Gas, Wasser) | **Resonanz in der Wassersäule** → Verstärkte Strömungen |
| **Tiefenbeben** | < 0.1 Hz | Magmaaufstieg in großer Tiefe | **Lokale Magnetfeldschwankungen** (piezoelektrische Effekte) |
| **Oberflächenwellen (Rayleigh/Love)** | 0.01–0.1 Hz | Tektonische Verschiebungen | **Veränderung der Windschubspannung (τ)** → Störung der Ekman-Spirale |
| **Infraschall** | < 20 Hz | Explosionen, Gasausbrüche | **Druckgradienten in der Atmosphäre** → Windänderungen |

**Quelle**: [USGS Volcano Seismology](https://www.usgs.gov/observatories/hvo/volcano-seismology)

---

### **B) Vulkanische Aktivität und Magnetfeldschwankungen**
#### **1. Piezoelektrische Effekte**
- **Mechanismus**:
  - **Gesteine (z. B. Basalt)** sind piezoelektrisch – sie erzeugen **elektrische Ladungen** unter Druck.
  - **Vulkanische Beben** → **Druckänderungen im Untergrund** → **lokale Magnetfeldanomalien**.
- **Messdaten**:
  - **Beispiel: Ätna (Italien)**:
    - Während Eruptionen wurden **Magnetfeldschwankungen von bis zu 10 nT** gemessen ([Del Negro et al., 2002](https://agupubs.onlinelibrary.wiley.com/doi/10.1029/2001JB000659)).
  - **Beispiel: Kīlauea (Hawaii)**:
    - **Magnetfeldänderungen von ~5 nT** während seismischer Krisen ([Zlotnicki et al., 2001](https://agupubs.onlinelibrary.wiley.com/doi/10.1029/2000JB000048)).

#### **2. Thermische Effekte**
- **Heiße Magma** → **Lokale Erwärmung des Meerwassers** → **Änderung der elektrischen Leitfähigkeit (σ)**.
- **Folge**:
  - **Verstärkte Lorentz-Kraft** in der Ekman-Schicht (da \(F_L \propto \sigma\)).

#### **3. Elektromagnetische Induktion durch vulkanische Ströme**
- **Magma enthält geladene Teilchen** (Ionen, Elektronen).
- **Bewegung des Magmas** → **elektrische Ströme** → **sekundäres Magnetfeld**.
- **Folge**:
  - **Lokale Magnetfeldgradienten** in der Nähe des Vulkans.

**Fazit**:
- **Vulkanische Seismik kann Magnetfeldschwankungen von 5–10 nT verursachen** – genug, um die **Lorentz-Kraft in der Ekman-Schicht zu verstärken**.

## **2. Vulkanische Aktivität und Ozeanströmungen**
### **A) Druckwellen und Turbulenzen**
- **Vulkanische Tremor (1–10 Hz)**:
  - **Druckwellen breiten sich im Wasser aus** → **Turbulenzen in der Ekman-Schicht**.
  - **Folge**:
    - **Verstärkte Durchmischung** → **kühleres Oberflächenwasser** (Upwelling).
    - **Störung der Ekman-Spirale** → **ungewöhnliche Strömungsmuster**.
- **Langperiodische Erdbeben (0.1–1 Hz)**:
  - **Resonanz in der Wassersäule** → **verstärkte Strömungen**.
  - **Beispiel**:
    - **2018 eruption of Anak Krakatau** → **Tsunami in der Sundastraße** durch submarine Hangrutschungen ([Grilli et al., 2019](https://www.nature.com/articles/s41598-019-40146-1)).

### **B) Wärmefluss und Upwelling**
- **Heiße Lava/Asche im Meerwasser** → **lokale Erwärmung** → **Konvektion**.
- **Folge**:
  - **Auftrieb warmer Wassermassen** → **Veränderung der Dichtegradienten**.
  - **Störung der Ekman-Spirale** → **Linksdrehung möglich**.

### **C) Windänderungen durch vulkanische Aktivität**
- **Aschewolken und Gasausbrüche** → **lokale Erwärmung der Atmosphäre** → **Änderung der Druckgradienten**.
- **Folge**:
  - **Verstärkte oder abgeschwächte Trade Winds** → **direkte Störung der Ekman-Spirale**.

**Fazit**:
- **Vulkanische Aktivität kann Ozeanströmungen auf mehrere Weisen stören**:
  1. **Direkte Druckwellen** → Turbulenzen.
  2. **Wärmefluss** → Upwelling/Downwelling.
  3. **Windänderungen** → indirekte Wirkung auf die Ekman-Spirale.

## **5. Physikalische Modelle: Wie könnte der Mechanismus funktionieren?**
### **A) Vereinfachtes Modell der Wechselwirkungen**
```
Vulkanische Seismik
   ↓
Druckwellen / Wärmefluss / Windänderungen
   ↓
Magnetfeldschwankungen (5–10 nT) + Turbulenzen in der Ekman-Schicht
   ↓
Verstärkte Lorentz-Kraft + gestörte Coriolis-Kraft
   ↓
Linksdrehung der Ekman-Spirale + kühlere Karibik
```

### **B) Mathematische Beschreibung**
1. **Magnetfeldänderung durch Seismik**:
   \[
   \Delta B = k \cdot \Delta P
   \]
   - \(\Delta P\): Druckänderung durch vulkanische Aktivität.
   - \(k\): Proportionalitätsfaktor (abhängig von Gesteinseigenschaften).

2. **Lorentz-Kraft in der Ekman-Schicht**:
   \[
   F_L = \sigma \cdot (\vec{v} \times \vec{B}) \times \vec{B}
   \]
   - **Wenn \(\Delta B\) lokal erhöht ist**, steigt \(F_L\) → **Linksdrehung möglich**.

3. **Strömungsänderung durch Turbulenzen**:
   \[
   \text{Turbulente kinetische Energie} = \frac{1}{2} \rho \cdot v_{\text{rms}}^2
   \]
   - \(v_{\text{rms}}\): Wurzel aus mittlerem Quadrat der Geschwindigkeit.
   - **Folge**: **Verstärkte Durchmischung** → **kühleres Oberflächenwasser**.

## **8. Python-Code: Simulation der vulkanisch induzierten Ekman-Spirale-Störung**
Hier ein **erweitertes Modell**, das **seismische Druckwellen und Magnetfeldänderungen** berücksichtigt:

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.signal import chirp

# Parameter
omega = 7.2921e-5  # Erdrotationsrate (rad/s)
sigma = 4.0        # Leitfähigkeit (S/m)
B0 = 40e-6          # Hintergrund-Magnetfeld (T)
v0 = 0.1            # Normale Strömungsgeschwindigkeit (m/s)
depth = 100         # Tiefe der Ekman-Schicht (m)
dz = 1              # Tiefenschritt (m)
seismic_amplitude = 10e-9  # Druckwellen-Amplitude (Pa) - typisch für vulkanischen Tremor
seismic_freq = 5    # Frequenz der Druckwelle (Hz)

# Vulkanische Druckwelle (vereinfacht als Sinuswelle)
def seismic_wave(t):
    return seismic_amplitude * np.sin(2 * np.pi * seismic_freq * t)

# Coriolis-Kraft
def coriolis_force(v, omega):
    return 2 * omega * v

# Lorentz-Kraft (mit vulkanischer Verstärkung)
def lorentz_force(v, B, sigma):
    return sigma * v * B**2

# Ekman-Spirale mit vulkanischem Einfluss
def ekman_spiral_with_volcano(depth, dz, omega, sigma, B0, v0, seismic_amplitude, seismic_freq):
    z = np.arange(0, depth, dz)
    u = np.zeros_like(z)
    v = np.zeros_like(z)
    angle = np.zeros_like(z)
    B = np.zeros_like(z) + B0  # Hintergrundfeld

    for i in range(1, len(z)):
        # Vulkanische Druckwelle beeinflusst Magnetfeld lokal
        B[i] = B0 + seismic_wave(i * dz / 1000) * 1e6  # Annahme: 1 nT pro 1 µPa Druckwelle

        # Coriolis-Effekt
        u[i] = u[i-1] - coriolis_force(v[i-1], omega) * dz
        v[i] = v[i-1] + coriolis_force(u[i-1], omega) * dz

        # Lorentz-Kraft (kann nach links oder rechts wirken)
        if B[i] > B0:  # Verstärktes Magnetfeld wirkt nach links
            u[i] += lorentz_force(v[i-1], B[i], sigma) * dz
        else:
            v[i] += lorentz_force(u[i-1], B[i], sigma) * dz

        # Winkel berechnen
        angle[i] = np.degrees(np.arctan2(v[i], u[i]))

    return z, u, v, angle, B

# Simulation
z, u, v, angle, B = ekman_sp
